Tips for beginner devs

  • You can find single file games on the ultimate game stash.
  • Keep your site optimized for slower devices and avoid heavy javascript use.
  • Keep backups and be prepared, some freehosts like neocities might remove your site and you might face other problems later on.
  • Try to keep it unblocked, add tools and extra features like about:blank and cloaking and not just dump a bunch of games.
  • update it often and use a contact option to get user feedback.


Unblocked Games rating

The rating system was originally made for my own site but is now being used for sites on UBGdir as a way to tell how good a site is, its not perfectand might be inaccurate. heres how it works.
  • one game = 1 ubg rating point
  • one proxy (external links not counted) = 5 ubg rating points
  • one additional feature = 3 ubg rating points
  • one anti block feature (such as about:blank loading or tab cloaking) = 3 ubg rating points
  • i might add extra points (max 50) for things that are not listed (rare)
